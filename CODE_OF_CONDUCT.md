# Code of Conduct

## Our Standard

This project expects respectful, professional collaboration.

Examples of expected behavior:

- constructive technical feedback
- respectful disagreement
- clear, reproducible issue reports

Examples of unacceptable behavior:

- harassment or abusive language
- discriminatory comments
- deliberate disruption of project workflow

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.
