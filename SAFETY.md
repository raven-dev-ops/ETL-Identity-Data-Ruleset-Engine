# Safety and Data Scope

This repository is for synthetic ETL identity-resolution experimentation only.

## In Scope

- synthetic person/incident-like records generated from first principles
- pipeline logic for normalization, matching, and survivorship
- quality reporting on synthetic outputs

## Out of Scope

- production integrations
- criminal justice information (CJI)
- agency operational records
- personally identifying real-world data

## Repository Rule

Do not commit any non-synthetic record data to this repository.
